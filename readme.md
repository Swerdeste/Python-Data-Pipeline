## Installation

1. Cloner le dépôt

```
git clone  https://github.com/Swerdeste/Python-Data-Pipeline.git
cd Python-Data-Pipeline
```

2 . Créer un environnement virtuel

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

3 . Lancer le script

```
python main.py
```
